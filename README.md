# ESP32-Based-Bluetooth-Controlled-Car
ESP32-based Bluetooth controlled robotic car that connects to a smartphone for wireless navigation. Built using ESP32, L298N motor driver, 4 gear motors, 4-wheel plastic chassis, and dual 18650 batteries for efficient and responsive movement control.

🚗✨ Excited to share my latest IoT project – an **ESP32-Based Bluetooth Controlled Car**!

I designed and built a smart robotic car that can be controlled directly from a smartphone via Bluetooth. The ESP32 acts as the brain of the system, receiving commands from the mobile application and controlling the movement of the car in different directions.

🔹 **Key Features:**
✅ Wireless control using Bluetooth connectivity
✅ Forward, Backward, Left, Right, and Stop movements
✅ Compact and low-cost robotic car design
✅ Easy to upgrade with additional sensors for obstacle avoidance and autonomous navigation

🛠️ **Components Used:**
• ESP32 Development Board
• L298N Motor Driver Module
• 4 × Gear Motors
• 4 × Wheels
• Plastic Car Chassis
• 2 × 18650 Rechargeable Batteries
• Jumper Wires and Connecting Accessories

This project helped me gain practical experience in:
🔸 Embedded Systems Programming
🔸 ESP32 and Bluetooth Communication
🔸 Motor Control using L298N
🔸 Robotics and IoT System Integration

Looking forward to enhancing this project further by integrating ultrasonic sensors, line-following capabilities, and autonomous navigation features.
